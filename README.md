# Skylark Drone Intelligence System

## Overview
This project simulates a drone-based aerial intelligence platform that analyzes surveillance events and generates actionable insights using an AI-driven agent.

## Features
- Drone path visualization
- Event monitoring on map
- AI-based risk analysis
- Dynamic summary generation
- Human-in-the-loop approval system

## Tech Stack
- Frontend: React
- Backend: Spring Boot
- Deployment: Vercel + Render
